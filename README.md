#### 注意
> 在**Bootstrap**中，栅格布局系统中的每一行(row)被分为**12**列。


#### (列)类前缀
* `.col-xs-`: xs =&gt; *X Small* 超级小的 **手机**
* `.col-sm-`: sm =&gt; *Small* 小的 **平板**
* `.col-md-`: md =&gt; *Middle* 中等的 **笔记本**电脑
* `.col-lg-`: lg =&gt; *Large* 大的 **台式机**电脑

#### 列占用
> 宽度值得是列宽(N)，取值范围为1到12之间的整数。
* 列宽: `.col-md-N`
* 列偏移: `col-md-offset-N`
